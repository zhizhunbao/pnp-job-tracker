模块 Banner 设计总表(2026-07-19)—— 图片怎么找 / 怎么轮 / 放什么信息 / 尺寸规范

核心原则四条:① 宽高全固定(banner 桌面恒 150px 高、1320 轨内全宽;手机 110px;所有图 object-fit:cover 中心裁剪,不随内容抖)——你问的「宽高应该固定吧」:对,全站图片位一律定框裁图,已定框的:news 头条图区 240px、列表缩略 96×64,banner 加入 150px 一档; ② 两类轮播分清:A类=内容轮播(news 头条那种,图+文一起换,5s+箭头+圆点+hover 暂停,已上线);B类=氛围背景轮播(四模块 banner 用,只有背景图 crossfade 淡换,前景信息恒定不动,8s 慢速、无箭头、右下小圆点,信息永远可读); ③ 图片来源只用 Wikimedia Commons 实景(续 /news 惯例:Special:FilePath 1200px 缩略下载 → 裁成 1280×300 存 cms/public/img/banners/,SOURCES.md 记出处,致谢挂 img title;不外链、不用抓来的文字图——文字图教训); ④ 信息=模块名+一句定位+2-3 个实时数字+右槽 CTA(数字全部来自页面现有 SSR 数据,零新查询);缺图/加载失败自动退现浅色渐变带(现 PageBanner 即兜底态,零风险上线。)

规范表(实装口径)

项规范
尺寸桌面:1320 轨内全宽 × 恒 150px;手机 恒 110px(数字收到 2 个,CTA 换行藏);图源文件统一裁 1280×300 再入库,object-fit:cover 兜底
图片来源Wikimedia Commons(Special:FilePath 1200px 下载→裁 1280×300→cms/public/img/banners/<module>-1..3.jpg;SOURCES.md 记出处;致谢=img title,画面无水印)。每模块 3 张起步
轮播(B类)背景 crossfade 淡换,8s;前景文字/数字/CTA 恒定不动;右下小圆点(可点跳),无箭头;hover 暂停;prefers-reduced-motion 时静止首图
信息模块名 + 一句定位 + 2-3 个实时数字(全取页面现有 SSR 数据,零新查询)+ 右槽 CTA(jobs=建档 / pathways=建档 / rank=今日榜 / stats=跨省对比;news 无)
对比度红线模块色渐变暗化层压图(左浓右淡),文字带 text-shadow;任何图上白字对比度 ≥ 4.5:1
兜底图缺/加载失败 → 退现浅色 PageBanner(现状即兜底,发布零风险);PageBanner 组件加 images? 可选参,不传=现状不变
实施顺序建议:先 jobs+news 文案态(零图可上)→ 选图裁图一次备齐 12 张 → 四模块开图。过目后回「按这个做」/「改 X」。
