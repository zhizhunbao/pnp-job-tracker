# E8-03 · 移动自适应一揽子(所有组件手机可用)

> Epic **E8 UI 体验统一** · 负责人 Frank · 4 SP · 批次 B10
> 通用约定与索引见 [实现文档 README](../README.md)。2026-07-06 用户拍板:「所有的组件都要满足自适应,方便在手机上也能观看」。

---

## 1. 整体目标

全站(/jobs 主页 + 所有弹窗 + E8-02 弹窗化后的功能)在 **375px 视口**(iPhone 基准)可读可操作:不横向溢出页面、点击目标不小于 40px、弹窗窄屏自动全屏。桌面布局不回退。

## 2. 验收标准

- [x] /jobs:顶栏/横幅/筛选区/搜索行 375px 不换行错位(顶栏两段 flex-wrap,右组去 flexShrink:0——它曾把页面撑出 31px);表格在窄屏可横滚(冻结列保留)且页面本体不横向滚动(实测溢出 0px)。
- [x] 所有弹窗:≤640px 视口自动全屏(Modal 壳统一;顾问弹窗强制全屏、禁拖拽/八向拉伸/全屏钮);内边距收窄档进 Modal.tsx。
- [ ] 筛选区窄屏折叠(常用筛选默认收起为一行「筛选」开关,展开抽屉式)——现状纵排可用,折叠优化待做。
- [ ] playwright 375/768/1280 三档截图走查清单过一遍(每页每弹窗)——已过:/jobs 375 + JD 弹窗 + 顾问弹窗;其余页面待走查。
- [ ] 触控:表头排序/列宽拖拽在触屏不误触(拖拽手柄 pointer events 已用 Pointer API,验证 touch)。
