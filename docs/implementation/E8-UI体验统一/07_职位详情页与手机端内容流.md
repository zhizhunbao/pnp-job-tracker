# E8-07 职位详情页与手机端内容流(内容站骨架借鉴批)

> 缘起:2026-07-20 Frank 看内容站手机端后拍板「排版和交互可以借鉴,配色和其他不用」。
> 借鉴对象=标准**单栏内容详情页骨架**(标题→meta→面包屑→分节正文→标签→上下篇→相关推荐)+手机端三件(常驻搜索/全屏抽屉导航/单列卡片流)。
> **配色红线:全站现行浅色体系(白卡+描边+#2563eb 蓝)原样沿用,禁暗色禁荧光。**
> 本件=设计文档(molit 惯例开工前出);**未获批不动代码**。

## 1. 目标与非目标

- 目标:
  - ① **`/jobs/<id>` 独立职位详情页**(A 件,本批核心):单栏内容骨架,SSR 直出。桌面弹框照旧不动,详情页=可分享 URL+SEO 落地页+手机端主阅读形态;
  - ② **手机端一岗一卡**(B 件):窄屏 jobs 主表由横滚表格换单列紧凑卡(职位名/公司/地点/薪资/发布时间+chips 一行),点卡直接进 ①——手机上不再弹框套弹框;桌面表格零改动;
  - ③ **手机端顶部常驻搜索条**(C 件):窄屏 sticky 顶栏下挂通栏搜索,不再收进筛选折叠区;
  - ④ **手机端侧滑抽屉导航**(D 件,2026-07-20 Frank 看图改拍侧滑式):汉堡钮开左侧约 4/5 宽抽屉(右侧露页面+遮罩),条目=圆角块,带子模块项 chevron 抽屉内展开二级(分组与桌面下拉一致);
  - ⑤ **SEO 落地**:详情页 generateMetadata(职位名+公司+城市)+ schema.org **JobPosting JSON-LD**(Google 求职富结果)+ sitemap 分片收录——两万余在库岗=两万余可收录落地页,直击「瓶颈在流量」;
  - ⑥ **顶栏 hover 下拉子模块**(E 件,2026-07-20 Frank 二批追加):桌面顶栏聚合项(资料库/资讯类)hover 即开单列下拉,分区区头小字+当前项高亮(现有资料库下拉对齐该模式,交互从点击改 hover,含键盘可达);
  - ⑦ **新闻评论区亮回 v2**(F 件,同批追加,Frank「新闻资讯板块完全参考」):#95 暂藏的评论区换新形态亮回——评论计数、**官方置顶楼**(admin 发+置顶标)、**楼中楼一层回复**(「展开 N 条回复」折叠)、列表页 💬 计数徽标(代码已有);登录+审核制照旧。
- 非目标:**不做暗色主题**;不做大图卡片流(职位无图,logo 占位=零信息);不做评论/广告位;不动桌面表格与弹框体系(重器保持);不新增抓取;не改筛选逻辑(卡片流复用现有 WHERE/分页)。

## 2. 红线落地

- **配色零新增**:全部用 primitives 现行 token(白卡 #fff、描边 #e5e7eb、蓝 #2563eb、灰注 #6b7280);Chip/Tag/Notice/Button 全复用,#114 后零新造组件。
- **W 规矩**:详情页 meta 行禁「·」「/」杂糅——公司、城市、发布时间各自成段(桌面同行分栏,窄屏换行);
- **代码不裸奔**:NOC 带职业名、省全名、TEER 带说明(详情页信息密度更大,更要人话);
- **分层不破口**:Pro 专属数据(vs 中位/匹配分)不进详情页 SSR 输出——免费可见字段与主表免费态完全同口径,Pro 维度=登录态卡内提示引导(⑤先例文案),真值仍走服务端 gate;
- **closed 岗**:详情页保留可访问(SEO 已收录不 404),顶部 Notice info「该岗已下架」+发布/关闭时间;JobPosting JSON-LD 带 validThrough。
- **隐私/顾问**:详情页内嵌 AI 顾问初判照走现行额度闸(#102 同款),无新口子;
- **评论不做匿名直发**:借鉴站=填个昵称就能发,那是灌水/中介广告敞口——本站保持 登录可评→pending→admin 审核→公开(Comments collection 现行访问控制原样),**信任边界不上砧板**;脱敏昵称快照照旧不漏邮箱;
- **评论只嵌一层**:楼中楼回复 parent 只允许指向顶层楼(禁无限嵌套,审核台和排版都可控)。

## 4. 交互 / 形态

- **详情页(单栏 max-width 860)**:标题 → meta 行 → 面包屑 → chips → 事实块 → JD 五节(整理版默认,看原文切换照旧)→ 顾问初判 → 通道节 → 相关职位 → 底部「返回职位板」;右上「在 Job Bank 打开 ↗」外链照旧;meta 行**公司名可点**=开公司弹框(与主表公司格同款);
- **手机卡片**:一卡四行(职位名加粗蓝链 / 公司蓝链 / 城市+薪资 / 时间+chips;**裸「知名」Tag 不上卡**——Frank 2026-07-20「只显示知名不清楚」,公司质量由 E12-08 公司分承接,落地后公司名旁挂分数),行高压紧对齐一亩三分地密度观感;收藏星标保留卡右上;**点击=分元素不整卡**(Frank 2026-07-20 拍板「公司名、职位名手机都可点」):职位名→详情页,公司名→公司弹框(K 懒探索照旧),卡身其余区域不响应防误触;
- **桌面入口**:主表职位格点击=弹框照旧;弹框标题栏加「打开完整页 ↗」小链(新标签),分享场景由此出;
- **搜索条(手机)**:sticky 顶栏(汉堡+logo+头像)下挂通栏搜索输入,提交=现行 ?q= 语义;
- **抽屉(手机)**:左侧滑入约 4/5 宽白底(右侧露页面加遮罩,点遮罩/×关),条目圆角块一项一行大点击区,当前页高亮;带子模块项(资讯/资料库)点 chevron 抽屉内展开二级缩进列表(▾/▸ 双态);
- **顶栏下拉(桌面)**:hover 开(离开 150ms 延时关防抖)、单列白卡描边、分区区头小字、当前项蓝底高亮;键盘 focus 同样可开(可访问性不上砧板);
- **评论区(news 详情尾部)**:「已有 N 条评论」计数头 → 官方置顶楼(蓝「官方」Tag,admin 帖)→ 顶层楼时间倒序 → 楼中楼缩进一档+「展开 N 条回复」折叠(默认收,≤3 条直接展开);输入框顶置:登录态=textarea+提交(发出后灰字「已提交,审核后显示」),未登录=一行「登录后参与讨论」链开登录框;审核制口径小注常驻。
