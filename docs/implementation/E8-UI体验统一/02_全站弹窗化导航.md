> Epic **E8 UI 体验统一** · 负责人 Frank · 3 SP · 批次 B10
> 通用约定与索引见 [实现文档 README](../README.md)。2026-07-06 用户拍板:「所有都应该是弹窗显示,不应该需要 返回这种按钮」「定价 也是弹窗的」。

## 1. 整体目标

/jobs 是唯一主页面,其余功能从 /jobs 出发一律**弹窗打开**(壳规范=[弹框规范](../../弹框规范.md)),不跳页、不需要「返回」。**SEO 例外原则**:stats/rankings/legal 这些有搜索流量价值的 URL(E5-04/E7-03 的 121 个 sitemap URL)**页面继续存在**供直链/收录,但站内导航不再跳过去——站内点击开弹窗,页面版只服务搜索引擎与外链。

## 2. 验收标准

- [ ] 定价:顶栏/锁点/横幅所有入口打开 **PricingModal**(内容=现 /pricing 的档位卡+按钮三态,plan.ts 单一来源);/pricing 页保留可直达(SEO/Stripe 回跳)。
- [ ] 榜单/统计:站内入口改弹窗(Modal md/lg,内容组件与页面版共用);/rankings/* /stats/* URL 照常渲染(sitemap 不动)。
- [ ] 账户:/account 三合一(状态/档案/退订)评估弹窗化;若表单过重可保留页面但去「返回」钮(浏览器返回即可)——决策记录在 §5。
- [ ] 全站无「返回/后退」按钮;弹窗一律 ESC/遮罩可关。
- [ ] 弹窗内容组件与页面版**同一份代码**(props 区分壳),不许 fork 两份。
