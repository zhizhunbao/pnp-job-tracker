## 1. 整体目标

账号相关入口目前散落三处:顶栏只有「登录」(AccountLink),「注册」按钮在未登录价值横幅里,「定价/Pro」在横幅链接与各锁点。归组为**顶栏右侧一个账户区**:未登录 = [登录] [注册] [Pro];已登录 = [用户名] [Pro 状态/升级]。横幅只留价值主张文案,不再重复按钮。

## 2. 验收标准

- [ ] 未登录顶栏右侧依次:登录 · 注册 · Pro(样式同组,视觉一体);点击分别开 AuthModal(login)/AuthModal(register)/定价弹窗(E8-02)。
- [ ] 已登录顶栏右侧:用户名(→ /account)+ Pro 态徽标或「升级」入口。
- [ ] ValueBanner 里的注册按钮/定价链接移除(横幅只留一句话+关闭钮),或整条横幅退役(用户定)。
