## 1. 整体目标

四个静态三语页面上线：免责声明全文、隐私政策、服务条款（含退款口径与内容下架机制）、关于页。这是敢收钱的正当性基础。

## 2. 验收标准

- [ ] `/legal/disclaimer` `/legal/privacy` `/legal/terms` `/about` 三语可访问，footer 链齐。
- [ ] 免责：非 RCIC / 信息仅供参考 / PNP 标记是粗筛非资格认定 / 以官方来源为准。
- [ ] 隐私：收集项（email、订阅状态、analytics 匿名统计）、支付由 Stripe 处理（本站不存卡号）、删号途径（邮件申请）。
- [ ] 条款：订阅与退款口径（如「随时退订、当期不退比例款」——D5 定价时一并拍板）、数据来源说明、**雇主异议下架机制**（收到异议 → 核实 → status=closed）。
