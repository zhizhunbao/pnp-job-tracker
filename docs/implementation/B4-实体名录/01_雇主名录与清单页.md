1. **名录页免费**(browse+搜索+SEO)——用户拿着心仪雇主来查「这家办过没有」,这是中介收 ¥5 万卖的信息;**转化钩=每行「查在招职位 →」深链 `/?q=<name>`**(既有 URL 参数回流,零新代码),落职位板后走既有「建档匹配」漏斗。

3. **数据全在库,零 schema 改动**:designated_employers 2,931 行(NS 1,574/NB 1,263/NL 94)· companies 带 LMIA 记录 3,788 行 · pnp_occupations 183 行。**LMIA 全量 7.5 万雇主(raw)留二期**——先用在库数据验证搜索流量再决定上全量(新表+DDL)。
4. **顶栏不动**(6 项已满):入口=SEO 直达 + SiteFooter 链接 + sponsor-likely 榜单页头交叉链;顶栏是否给「名录」留位=Frank 拍板项,不在本工作项。
