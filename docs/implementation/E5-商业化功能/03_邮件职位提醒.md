## 1. 整体目标

Pro 用户收到两类邮件：① 新岗命中我的路径（match 高分 + 可选 saved search）；② 新抽选 vs 我的 CRS。触发复用日更链路（seed 成功 → curl alerts run），**不加独立 cron**。

## 2. 验收标准

- [ ] Pro 用户可保存/删除筛选（上限 N 个，plan.ts 常量）；免费用户看到锁+升级引导。
- [ ] 日更后命中新岗（`first_seen > lastNotifiedAt` 且匹配 filters）→ 收到按用户语言的邮件（岗位表格+链接）；无命中不发。
- [ ] 同一岗不重复通知（lastNotifiedAt 回写）；alerts run 带 token 鉴权（无 token 401）。
