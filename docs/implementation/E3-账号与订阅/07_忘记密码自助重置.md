# E3-07 忘记密码自助重置

> 2026-07-17 凌晨 Frank 拍板「先来文档」——发帖(冷启动)前的客服止血:现状登录框写着
> 「忘记密码?公测期请邮件联系我们重置」= 每个忘密码用户都是一单人工客服;流量进来前改自助。
> 范式:**不自造轮子**(Ponytail 阶梯 3:平台特性)——Payload auth collection 自带
> `forgot-password / reset-password` 两个 REST 端点与 token 管理(生成/哈希/过期),缺的只是
> ① 邮件适配器(接现成 `lib/mailer.sendMail`,零新依赖)② 邮件模板指回前端 ③ 前端两个表单态。

## 1. 目标与非目标

- 目标:用户在登录框点「忘记密码?」→ 输邮箱 → 收邮件 → 点链接到本站设新密码 → 自动登录。全程零人工。
- **防枚举**:无论邮箱存在与否,提交后一律提示「若该邮箱已注册,重置邮件已发出」(Payload forgot-password 本身总回 200,前端文案配合)。
- 非目标:改密码强度策略(注册 minLength=8 不动)、邮箱验证(注册免验证的现状不动)、三语邮件模板(邮件正文 zh+en 双语一封,韩语后续看用户占比)。

## 2. 数据

- **零新表零新列**:Payload users 表自带 `reset_password_token` / `reset_password_expiration` 列(auth collection 建表时已生成,生产实查确认后勾验收①;若缺列则按 B7 手写 DDL 补齐再动代码)。
- token 有效期用 Payload 默认(1 小时,`auth.forgotPassword.expiration` 可调,不动)。

## 3. API(全部 Payload 内置,零自写路由)

- `POST /api/users/forgot-password` `{email}` → 200(总是);存在则写 token 并发信。
- `POST /api/users/reset-password` `{token, password}` → 成功返回登录态(带 set-cookie),前端整页刷新即已登录。
- **payload.config 两处新配置**:
  1. `email:` 自定义最小适配器包 `lib/mailer.sendMail`(Payload v3 接口:`{ name, defaultFromName, defaultFromAddress, sendEmail }`),**不装 @payloadcms/email-\* 包**;`RESEND_API_KEY` 未设时 sendMail 返回 false=静默 dry-run,与 alerts 同语义。
  2. Users collection `auth.forgotPassword.generateEmailHTML/Subject`:链接指 **前端** `${NEXT_PUBLIC_SITE_URL}/jobs?reset=<token>`(Payload 默认指 /admin 重置页,必须覆盖);正文 zh+en 双语、带 1 小时有效期说明与「非本人操作请忽略」。

## 4. 前端

- **AuthForm 登录 tab**:`acct.forgot` 死文案改为可点链接 → 切「找回密码」态(单邮箱输入框+提交);提交后固定提示 `acct.forgotSent`(防枚举文案)。
- **重置态**:/jobs 检测 `?reset=<token>`(与 ?login=1 同款钩子)→ 开 AuthModal 的「设新密码」态(新密码输入,minLength=8 挂上——这是注册同级的设置场景,#登录不做长度校验的拍板不冲突)→ POST reset-password → 成功 `history.replaceState` 洗参(#38 教训)+ 整页刷新(SSR 分层态生效,登录改版先例)。
- token 无效/过期 → 红字 `acct.resetBad` + 引导重发。
- i18n 新键(三语界面照旧,仅邮件正文双语):`acct.forgot`(改文案「忘记密码?」)/ `acct.forgotSent` / `acct.resetTitle` / `acct.resetBtn` / `acct.resetBad`。

## 5. 运维与安全

- 发信通道复用 Resend(域名已 Verified,`RESEND_FROM=alerts@offer2pr.com`);**@test.local 测试号收不到真信**——链路验证用 Resend 面板日志 + 本人邮箱实测一封。
- 频控:forgot-password 端点挂现有 `lib/rateLimit`(同 IP 5 次/小时),防轰炸他人邮箱。
- 老短密码账号(6 位)走重置后自然升到 ≥8 位,顺手消化 #登录 minLength 历史尾巴。
