# BC PNP Online User Portal

Last updated:** June 19, ****2026**

## Sign in

Candidates who wish to register or apply to the BC PNP must do so using the BC PNP Online system. To create an account or sign in, please click the button below.
[Open the BC PNP Online User Portal](https://www.pnpapplication.gov.bc.ca/user/sign-in)

## Technical support

If you are registering or applying to a Skills Immigration stream, see the [Skills Immigration Application Guide](https://www.welcomebc.ca/immigrate-to-b-c/about-the-bc-provincial-nominee-program/documents#For_workers) for step-by-step instructions.
