  About the Role  
 This position is for a Full-Stack Software Developer to join our team. You will be responsible for defining, designing, developing, and deploying enterprise and cloud native applications (such as  Solace Cloud ) in leading cloud providers – such as Amazon Web Services, Microsoft Azure, Google Cloud Platform – and on-premise (Kubernetes, Docker). 
   
  What You’ll Do    
 
 Deliver features as a full-stack developer  
 Communicate with the team through meetings, demos, and documentation 
 Test software through automation 
 Have a desire to continually learn new technologies 
 Have a desire to use AI technologies to accelerate their workflow 
 Contribute to new products/solutions using agile software development practices 
 
   
  What You’ll Bring to the Role  
 
 B.S. degree in Software/Computer/Electrical Engineering, Computer Science or similar field 
 Experience developing in back-end and front-end stacks 
 Experience with Java &amp; Spring Boot 
 Experience with REST APIs 
 Experience with databases (SQL, NoSQL) 
 Experience with JavaScript client frameworks (React) 
 Experience with Python 
 Comfortable working with various LLMs and AI tools (e.g. Copilot, Cline) 
 Comfortable with writing various types of automated tests 
 A self-starter, motivated, eager and willing to learn 
 A team player who can collaborate within cross-functional teams 
 A good communicator who communicates technical ideas clearly and concisely 
 
   
  Desirable Skills  
 
 Experience with cloud native infrastructure and SaaS 
 Experience with containers, microservices, and related technologies 
 Experience with event-driven architecture (EDA) and event brokers 
 Experience with CI/CD pipelines and technologies (GitHub Actions, Harness)
